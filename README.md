# EBBC impl
Unofficial EBEC component impl of Scene Text Recognition with Eliminating Background Noise and Enhancing Characters Shape Feature</br>
Use gpu with CUDA to run</br>
```bash
# To train
python ebec.py
# To validate
python test.py
```
